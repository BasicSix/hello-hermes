安装依赖

sudo apt install -y postgresql postgresql-contrib redis-server

🐘 PostgreSQL 配置

-- 创建数据库用户
CREATE USER hermes WITH PASSWORD 'your_secure_password_2026';
CREATE DATABASE hermes OWNER hermes;

-- 启用扩展
\c hermes
CREATE EXTENSION IF NOT EXISTS "uuid-ossp";
CREATE EXTENSION IF NOT EXISTS "pg_trgm";

-- 设置连接限制
ALTER USER hermes CONNECTION LIMIT 10;

🔴 Redis 配置

# /etc/redis/redis.conf
bind 127.0.0.1 ::1
port 6379
maxmemory 2gb
maxmemory-policy allkeys-lru
appendonly yes

🔗 Hermes 配置集成

memory_backend: postgresql connection_string: "postgresql://hermes:***@localhost/hermes" cache_backend: redis url: redis://localhost:6379/0

💡 性能建议:

  • 为 PostgreSQL 分配专用磁盘 I/O
  • Redis 使用内存大页（huge pages）
  • 定期清理 Redis 过期键
