🔒 基础安全

防火墙配置

# UFW 基础设置
sudo ufw default deny incoming
sudo ufw default allow outgoing

# 允许必要服务
sudo ufw allow OpenSSH
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp
sudo ufw enable

# 查看状态
sudo ufw status verbose

🛡️ 高级安全

SSH 加固

# /etc/ssh/sshd_config
Port 2222
PermitRootLogin no
PasswordAuthentication no
AllowUsers hermes
MaxAuthTries 3

应用安全头

# Nginx 安全配置
add_header X-Frame-Options "DENY";
add_header X-Content-Type-Options "nosniff";
add_header X-XSS-Protection "1; mode=block";
add_header Strict-Transport-Security "max-age=31536000; includeSubDomains" always;
add_header Content-Security-Policy "default-src 'self'";

🚫 Fail2Ban 配置

# /etc/fail2ban/jail.local
[DEFAULT]
bantime = 1h
findtime = 10m
maxretry = 3

[sshd]
enabled = true
port = ssh
filter = sshd
logpath = /var/log/auth.log
maxretry = 3

[nginx-http-auth]
enabled = true
port = http,https
filter = nginx-http-auth
logpath = /var/log/nginx/error.log

🔗 Hermes 安全集成

Hermes 提供密钥管理、配置加密与审计日志能力，与系统级安全无缝协同：

1️⃣ 敏感配置加密

# 使用 Hermes 内置加密保护 .env 文件
hermes encrypt --input ~/.hermes/.env --output ~/.hermes/.env.enc

# 启动时自动解密
hermes dashboard --decrypt-env ~/.hermes/.env.enc

2️⃣ API 密钥轮换（支持 OpenAI/LongCat）

# 自动轮换并更新 config.yaml
hermes rotate-api-key --provider openai --days 90

# 查看密钥使用统计
hermes api-stats --provider longcat

3️⃣ 审计日志启用

# 在 ~/.hermes/config.yaml 中添加
audit:
  enabled: true
  log_path: /var/log/hermes-audit.log
  level: high

⚠️ 重要: 启用审计日志会增加 I/O 开销，请确保 /var/log 有足够空间
